# Roadmap

## v0.2 (Current Upgrade Baseline)

- portable config auto-creation
- browser setup UI
- serial port discovery endpoint
- visual knob identify mode
- expanded docs and contributor guidance

## v0.3 (User-Facing Quality)

- one-click signed release artifacts
- better setup validation (port test button, live connection status)
- mapping templates (streaming, conferencing, music production)
- backup/restore config from UI
- localized UI strings (EN/IT)

## v0.4 (Audio Capability Expansion)

- implement `app` target (per-process volume)
- implement `group` target
- implement `mic_in` and `line_in`
- show available audio targets directly in setup UI

## v0.5 (Reliability and Maintainability)

- protocol versioning and compatibility checks
- integration tests with recorded serial fixtures
- firmware stress test for rapid encoder spins
- CI matrix builds for Windows/Linux/macOS
- release notes automation and artifact checksums

## Stretch Goals

- native desktop wrapper for setup UI (no browser dependency)
- firmware update tool inside setup UI
- profile switching with hardware long-press gestures
